## 🌟 Overview

**[AudioHD Pro]** is a powerful Chrome Extension designed for audiophiles and power users. Unlike standard volume boosters, it uses the **Web Audio API** and **AudioWorklets** to manipulate audio streams in real-time, providing studio-quality output even from low-bitrate sources.

Whether you are listening to **Spotify**, **SoundCloud**, **Apple Music**, or watching **YouTube**, this tool ensures your audio is crisp, deep, and perfectly balanced.

---

## ✨ Advanced Features

### 🔊 Premium Sound Control

* **10-Band Graphic Equalizer:** Precision control over frequencies from 32Hz to 16kHz.
* **Bass Booster & HD Crystalizer:** Deepen the low-end and sharpen the high-end for compressed audio.
* **600% Volume Boost:** Increase volume beyond system limits without distortion using our **Smart Peak Limiter**.

### 🌍 Universal Compatibility

* **Cross-Origin Capture:** Advanced CORS handling to work seamlessly on **Spotify, SoundCloud, and Apple Music**.
* **Shadow DOM Detection:** Deep-crawling technology to find hidden media elements in modern web apps.

### 🧠 Intelligent Logic

* **Bypass Engine:** A dedicated 'OFF' switch that routes audio directly to the output, ensuring zero latency when disabled.
* **Persistent Site Profiles:** Automatically saves and loads your custom EQ settings for each specific website.
* **Background Playback:** Optimized to keep processing audio even when you switch tabs.

### 🎨 Stunning UI/UX

* **Glassmorphism Design:** A modern, sleek, and dark-themed popup interface.
* **Live Audio Visualizer:** Real-time canvas-based animation that reacts to your music.

## 🚀 Installation (Developer Mode)

Since this is a custom-built tool, follow these steps to install it in your browser:

1.  **Download/Clone the Repository:**
   
    ```bash
    git clone [https://github.com/mmizan85/audiohd_pro.git](https://github.com/your-username/your-repo-name.git)
    ```

2.  **Open Chrome Extensions Page:**
   
    * Go to `chrome://extensions/` in your browser.
  
3.  **Enable Developer Mode:**
  
    * Toggle the **"Developer mode"** switch on the top right.

4.  **Load the Extension:**

    * Click on **"Load unpacked"**.
    * Select the folder where you cloned/extracted the files.
    * You should see **[AudioHD Pro]** in your list of extensions.
    * Make sure to allow permissions if prompted.

5.  **Pin & Play:**

    * Click the puzzle icon in Chrome and pin **[AudioHD Pro]**. Start any audio and enjoy!
